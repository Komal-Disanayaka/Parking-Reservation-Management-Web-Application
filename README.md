# ParkNest Smart Parking - Intelligent Parking Management System

A modern Spring Boot web application for intelligent parking management with user authentication and profile management.

## Features

- User Registration and Login
- User Profile Management (Update profile without changing username)
- User Profile Deletion
- MySQL Database Integration
- Spring Security Authentication
- Responsive Web Interface

## Technologies Used

- Spring Boot 3.2.0
- Spring Security
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
- Java 17

## Database Configuration

- Database: MySQL
- Username: root
- Password: 12345
- Database Name: parking_nest_db

## Setup Instructions

1. Make sure MySQL is running on your system
2. Create a database named 'parking_db'
3. Update application.properties if needed
4. Run the application using: `mvn spring-boot:run`
5. Access the application at: http://localhost:8080

## Default Port

The application runs on port 8080 by default.
